Export Data Package

 

After you have completed entering the EML metadata and uploaded all associated data files (data tables and other entities) to ezEML, Export Data Package creates a zip file containing the EML file (in both XML format and ezEML's JSON format) and the data files. This zip file will be downloaded to your browser's download directory. You can then submit it to the data repository of your choosing.


Note: The versions of the data files that will be incorporated in the zip file are the versions you've uploaded to ezEML. If you've modified a data file since it was uploaded, you will need to use Re-upload to provide ezEML with the updated version. Go to Data Tables and/or Other Entities to do any needed re-uploads. Then return to this page to Export Data Package.
