Export ezEML Data Package

 

When you have entered the EML metadata and uploaded all associated data files (data tables and other entities) to ezEML, you're ready to export the data package.

Export ezEML Data Package creates an archive file containing the EML file (in both XML format and ezEML's JSON format) and the data files. This archive file, with a '.ezeml.zip' file extension, can be downloaded to your computer or you can send its URL to EDI to submit the data package to the EDI data repository (see Submit to EDI).


Note: The versions of the data files that will be incorporated in the archive file are the versions you've uploaded to ezEML. If you've modified a data file since it was uploaded, you will need to use Re-upload to provide ezEML with the updated version. Go to Data Tables and/or Other Entities to do any needed re-uploads. Then return to this page to Export ezEML Data Package.
