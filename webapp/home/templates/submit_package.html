When you've entered all required metadata and uploaded all associated data files (data tables and other entities) to ezEML,#} {# you're ready to submit your data package to the data repository#} {# hosted by the Environmental Data Initiative (EDI).#}

Note: The versions of the data files that will be sent to EDI are the versions you've uploaded to ezEML. If you've modified a data file since it was uploaded, you will need to use Re-upload to provide ezEML with the updated version prior to submitting. Go to Data Tables and/or Other Entities to do any needed re-uploads. Then return to this page to Submit to EDI.

Note: To send your package to a recipient other than EDI (e.g., to send it to your LTER Information Manager), use Export ezEML Data Package. This will create an archive file containing your metadata and data files and provide you with a URL that points to the archive file. You can then email the URL to the target recipient, who can use it to download the archive to their local system. The archive can then be imported into ezEML, or it can be opened and processed manually (it contains the EML XML file and data files). For more details, see Sending Your Package to a Colleague in the User Guide.
