Submitting Your Data Package to EDI

 

After you have created and downloaded your EML metadata file, you may want to submit your data package to the data repository hosted by the Environmental Data Initiative (EDI). There are a couple of ways to do this.

The easier way is to use Export Data Package, which zips up your EML file and all of the data files comprising your data package and downloads the zip file to your browser's download directory. You can then send an email to support@environmentaldatainitiative.org with the zip file attached, requesting our assistance in publishing your data package in the EDI repository. We will be glad to help.

Alternatively, you can submit the data package yourself via the EDI Data Portal. You will need to log in there, then go to "Evaluate/Upload Data Packages" under the "Tools" menu. You will want to select the option to manually upload data from your local file system. If you have never used the Data Portal before, we will be happy to walk you through the process. Just email us at support@environmentaldatainitiative.org.
