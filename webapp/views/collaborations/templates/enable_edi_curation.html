Submit Package to EDI

 {{ macros.help_button(help_enable_edi_curation_btn) }}

Clicking Submit Package to EDI below will hand your ezEML data package off to EDI's data curation team, allowing them to access it directly. They will review it, and if any changes or corrections are needed, they will contact you before changing anything. When all needed changes have been made, they will submit the package to the EDI data repository. Note that this process may take several days.

If you have questions, please contact us at support@edirepository.org.
